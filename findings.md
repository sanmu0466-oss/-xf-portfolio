# 研究发现

## 当前作品集问题

1. **视觉同质化**：暖黄底(#f8f6f1) + 深蓝(#1e3a5f) + 金色(#b8942d) = AI 生成作品集三大模板之一（见 frontend-design SKILL.md 明确指出的反模式）
2. **交互单一**：纯静态，仅 lightbox
3. **信息过载**：单页堆砌所有内容
4. **移动端弱**：仅基础 grid 降级
5. **缺乏品牌感**：未传达政务自动化的专业/信任/现代感

## 设计参考

- frontend-design SKILL.md: 不要用暖黄+村线+陶土色、近黑+荧光绿、报纸式排版这三个默认模板
- ui-ux-pro-max SKILL.md: 优先级1=无障碍(对比度4.5:1)、优先级2=触控(44×44px最小值)、优先级4=风格一致性
- Precision UI 路线：冷灰白底 + 单一品牌色 + 无衬线 + 数据可视化精度感
